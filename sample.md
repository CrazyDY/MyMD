# 欢迎使用 Markdown Editor

这是一个**简单**而*强大*的 Markdown 编辑器。

## 功能特点

### 1. 实时预览
在左侧编辑，右侧即时看到渲染效果。

### 2. 代码高亮
```python

    print("Hello, Markdown!")
```

## 引用

> Markdown 是一种轻量级标记语言，
> 它允许人们使用易读易写的纯文本格式编写文档，
> 然后将其转换成有效的 XHTML（或者 HTML）。
